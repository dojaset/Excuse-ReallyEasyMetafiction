## ✍️ 요약
- 대학 재학 중 Java로 제작, 막 프로그래밍을 배우기 시작한 시기의 작업물

- **Java, Eclipse** - 프로그램 구현
- **ClipStudio Paint** - 일러스트 작업

## 📌 배운 점
- 당시 작성한 코드를 보면 자신의 프로그래밍 실력이 얼마나 발전했는지 알 수 있음
- 본격적으로 프로그래밍을 배우려 마음먹은 시기의 첫 작업물이라는 점에서 큰 의미
